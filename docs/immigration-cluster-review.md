## 발행 전 반드시 사람이 확인할 사실 (근거표 기준)

### 🔴 RED — 매 발행 주기 사람 검증 필수 (틀리면 신뢰·법적 리스크)
- **H-1B $100,000 수수료 (`h1b-100k-fee-explained`)** — 2025년 9월 포고령. 확인 필요:
  - 시행 시점(어떤 청원부터 적용되는 기준일)
  - **미국 내 신분변경(change of status) 케이스의 적용 여부** — 초안은 "제외될 수 있으나 확정 불가, 변호사 확인 필수"로 서술. 최신 USCIS 지침·소송 결과로 재확인.
  - 진행 중 소송에 따른 규정 정지/수정 가능성
  - → 초안은 확정 단정을 피하고 "마지막 업데이트: 2026년 7월"을 명시했으나, **발행 시점의 최신 상태로 갱신**할 것.
- **Visa Bulletin 문호(우선일자)** — `eb3-employment-green-card`, `nurse-eb3-schedule-a`에서 언급.
  - 초안은 **특정 날짜를 하드코딩하지 않음**(국가별 매월 변동, 한국=Rest of World 여유 정도만 서술). 발행 시 당월 국무부 Visa Bulletin과 모순 없는지 확인.

### 🟡 YELLOW — 수수료/숫자 최신값 확인 (근거표 갱신 시 자동 반영)
근거표(`facts-immigration.json`)의 아래 값이 **발행 시점 기준으로 여전히 맞는지** 확인:
- SEVIS I-901: **$350**
- MRV 비자 신청 수수료: **$185**
- H-1B 캡: **85,000** (일반 65,000 + 미국 석사 20,000)
- H-1B 전자 등록(registration) 수수료: **$215** (FY2026 인상값 — 매년 변동, 특히 확인)
- STEM OPT: **24개월** 연장 / 표준 OPT **12개월** / 합계 **36개월**
- 실업 허용일: 표준 **90일** / STEM 연장 시 총 **150일**
- H-1B 유효: 최초 **3년**, 최대 **6년**
- H-1B 누적 추첨 확률 **25% → 44% → 58%**: **모델 추정치**(실측 아님). 초안에 "참고치"임을 명시했으나, 오해 소지가 있으면 삭제 가능.

### 🟢 GREEN — 절차/개념 서술 (안정적, 경미한 표현 검토만)
- EB-3 3분류(숙련/전문/비숙련), PERM→I-140→I-485 3단계
- RN·물리치료사의 Schedule A Group I, VisaScreen, NCLEX-RN

## 발행 방법 (검토 완료 후)
1. 근거표 값이 최신인지 확인(위 YELLOW/RED). 바뀌었으면 `facts-immigration.json`을 먼저 갱신하고, 해당 초안 본문 숫자도 맞춘 뒤 `node scripts/check-facts.js`로 재확인.
2. 발행할 파일의 프론트매터를 `status: ready`, `enhancedDate: '2026-07-13'`(검토일)로 변경. (RED 2건은 최신 확인 전까지 draft 유지 권장.)
3. `node scripts/build.js` 통과 확인 → commit → push. 월요일 09:00 KST 드립이 발행.
4. `visa` 카테고리에 published 글이 처음 생기면 자동으로 `/blog/visa/index.html`과 사이트맵 항목이 생성됩니다(빌드가 카테고리 무관하게 처리).
